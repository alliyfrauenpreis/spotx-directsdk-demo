eachVPAIDEvent(iteratee, context)

Iterates over every event listed in spotx.iab.VPAID.VPAID2Event and calls iteratee(eventName). The iteratee is bound to context

The DirectAdOS Will publish the following VPAID events:

  • AdLoaded
  • AdStarted
  • AdStopped
  • AdSkipped
  • AdSizeChange
  • AdRemainingTimeChange
  • AdVolumeChange
  • AdVideoStart
  • AdVideoFirstQuartile
  • AdVideoMidpoint
  • AdVideoThirdQuartile
  • AdVideoComplete
  • AdClickThru
  • AdUserMinimize
  • AdUserClose
  • AdPaused
  • AdPlaying
  • AdError
Parameters:
Name Type Description
iteratee function
context

the this binding for iteratee
Example
var sdk = new SpotX.DirectAdOS(options)
sdk.eachVPAIDEvent(function (eventName) {
     console.log("Publisher :: Requesting Event Subscription: " + eventName);
     sdk.subscribe(function() {
         console.log("Received an Event!");
     }, eventName);
 }, this);

loadAd()

Loads ad unit and returns compliant Promises/A+ promise

Returns:

compliant Promises/A+ promise

Example
var options = ...;
    var sdk = new SpotX.DirectAdOS(options)
    sdk.loadAd().then( function() {
        console.log("Ad Unit loaded.");
      }, function() {
        console.log("Failed to load Ad Unit");
      });

loaded() → {Boolean}

Returns true only when both are true:

  • Ad unit has been successfully loaded
  • Ad unit has not finished playing (it is getting unloaded after playing)
